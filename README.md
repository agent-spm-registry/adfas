## Overview

This is an SPM skill collection — a package containing multiple skills.
Consumers can install individual skills or the entire collection.

## Skills Included

| Skill | Type | Description |
|-------|------|-------------|
| skill-md | rules | Skill "skill-md" from adfas collection |


## Install

### Install the entire collection
```bash
spm pull adfas
```

### Install a specific skill
```bash
spm pull adfas/<sub-skill-name>
```

## Adding More Skills

1. Create a new directory under `skills/`
2. Add an `spm.json` and `SKILL.md` in the new directory
3. Run `spm validate` to check everything
4. Run `spm push` to publish the updated collection
